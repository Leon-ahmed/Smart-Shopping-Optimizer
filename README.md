# Smart Shopping Assistant 🛒

A simple web project that helps users sort products and choose the best items within a budget.

---

## Features

- Sort products by rating (Quick Sort)
- Sort by price low to high (Merge Sort)
- Sort by price high to low (Merge Sort)
- Select products and set a budget
- Get best product combination using Knapsack algorithm

---

## Technologies Used

- HTML
- Tailwind CSS
- JavaScript

---

## Algorithms Used

- Quick Sort → for rating sorting
- Merge Sort → for price sorting
- Knapsack → for budget optimization

---

## How It Works

1. Products are displayed on the page  
2. User selects sorting option  
3. User selects products and enters budget  
4. System finds best products within budget  
5. Result is shown in optimized cart  

---

## Purpose

This project shows how algorithms like sorting and knapsack can be used in real shopping systems.
